Chart Embed API

Embed styled charts into any page with a single div and a JSON config. Fonts, colors, tooltips, and responsive behavior are handled automatically.

Quick Start

Add the embed script to your page, then place chart divs wherever you want charts:

<!-- 1. Add the embed script (once per page) -->
<script src="https://gorgias-ecom-lab-benchmark-web.vercel.app/embed.js" defer></script>

<!-- 2. Add a chart -->
<div data-gorgias="chart"
  data-chart-type="bar"
  data-chart-config='{
    "title": "Adoption by GMV",
    "data": [
      { "label": "$1M", "value": 18.7 },
      { "label": "$5M", "value": 22.1 },
      { "label": "$10M", "value": 28.4 }
    ],
    "yAxis": { "format": "percent" }
  }'>
</div>

Chart Types

data-chart-typeDescriptionBest for
barVertical bar chartComparing categories, rankings
lineLine chart with dot markersTrends over time
areaLine with gradient fillVolume trends
multi-lineMultiple named linesComparing series

Config Reference

Shared options (all chart types)

PropertyTypeDefaultDescription
titlestring—Chart title
subtitlestring—Smaller text below title
sourcestring—Attribution line below chart, e.g. "Gorgias platform data · March 2026"
heightnumber300Chart height in pixels
colorsarray of stringsPastel paletteOverride colors (hex values)
gridbooleantrueShow horizontal grid lines
legendbooleantrue for multi-lineShow legend
xAxisAxisConfig—X-axis configuration
yAxisAxisConfig—Y-axis configuration

AxisConfig

PropertyTypeDescription
format"number" | "percent" | "currency" | "minutes" | "hours"Auto-formats tick labels
formatTemplatestringCustom template, e.g. "{value} pts"
hidebooleanHide the axis
domain[min, max]Set axis range

Single-series options (bar, line, area)

PropertyTypeDescription
data{ label, value, color? }[]Required. Array of data points
splitIndexnumberBar only: two-tone split point
curve"monotone" | "linear" | "step"Curve interpolation
dotsbooleanShow point markers

Multi-line options

PropertyTypeDescription
series{ key, label, color?, data }[]Required. Named data series
curve"monotone" | "linear"Curve type for all lines
dotsbooleanShow point markers (default: true)

Card: White background, rounded corners, responsive padding

Tooltips: Styled automatically on hover

Responsive: Charts resize at 320px, 568px, and 768px breakpoints
